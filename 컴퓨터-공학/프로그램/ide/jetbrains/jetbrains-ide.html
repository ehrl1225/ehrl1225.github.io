단축키

  • 전체 검색 : shift + shift
  • 코드 검색 :
    • mac : shift + command + F
    • windows: ctrl + shift + F
    • 검색에서 제외하기
      • 디렉토리 마크해서 제외하면 됨
  • 코드 자동 정렬
    • mac : command + option + L
    • windows : ctrl + alt + L
  • import 정리
    • mac : control + option + O
    • windows : ctrl + alt + O
  • Commit
    • mac : command + K
    • windows : ctrl + K
  • Push
    • mac : shift + command + K
    • windows : shift + ctrl + K
